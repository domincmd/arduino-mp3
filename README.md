# arduino mp3
this is a project for a simple mp3 player, the code for the arduino is embedded.

## components
this project was built using: 
 - arduino nano
 - dfplayer mini
 - 2 push buttons
 - headphone jack
 - one 1kOhm resistor
 - 9v battery
 - one switch (on/øff)

## technical stuff
this project was programmed using the DFRobotDFPlayerMini library, and made good use of the only 2 push buttons it features.

to pause/play the song, click one button. to toggle between folder/file skipping, double click the button 
to skip to the next song/folder, press the other button. to go back, hold the same button until the song/folder changes.
